# React Exam - May 2026

### Instructions

This repository contains the starting code for a lab-based exam for React app development. You are required to take the following steps in preparation for this exam:

- Clone this repository.
- Open the project into VS Code.
- Create a new **private** repository in your GitHub account called '**reactExam**'.

- In VS Code terminal, type the following commands:

```
git remote remove origin
git remote add origin ...URL of your new repository...
git push origin main
```



- Run the following commands:

```
npm install
npm run dev
```



Now check that the App runs correctly and is accessible from a browser on your machine.
**Please ensure you have completed these steps before the lab-based exam and that you can push changes to your remote repository on GitHub.**

### API

The app uses the public Open Library API. No API key is required for the starter flow.

### Routes

- `/` - Home page with the book list
- `/books/:id` - Book details page

### Project Structure

- `src/api/` - Central API helpers
- `src/components/` - Reusable UI components
- `src/pages/` - Route-level pages
- `src/types/` - Shared TypeScript types
- `src/images/` - Static images such as the placeholder book cover

This repo is designed as a baseline for an in-lab React assessment.
